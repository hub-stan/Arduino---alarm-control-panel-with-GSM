# Arduino---alarm-control-panel-with-GSM

In code not implemented possibilities: deactivate alarm and activate. This function can add ofcourse, so project is progressiv.

To build alarm need:

Sensor/limit switch
SIM800
Conecting wire
Sim cart able to send sms
